Javascript Intro

  • Javascript was called LiveScript in earlier days, due to JAVA popularity, it changes to Javascript, it has nothing to do with JAVA.
  • Js runs on browser to bring dynamic effects to the web page, like opening a modal in same page for brief info rather opening new page.
  • browsers have their own Javascript engine to compile js, google chrome uses -V8 engine, firfox uses - spider monkey.
  • V8 is very popular and used in Node Js as well.
  • Javascript versions are called with prefix ES - Ecma Script
  • ES5->ES6->ES2016->ES2017....ES2021
  • ES6 was the version had lots of changes, that is why it is called modern javascript from ES6 onwards
